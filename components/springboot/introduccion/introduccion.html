Introducción

Spring Boot makes it easy to create stand-alone, production-grade Spring based Applications that you can "just run".

La Anotacion @Component

El contenedor de Spring autodetecta las clases con la anotación @Component, lo que permite usar dichas clases como dependencias y así poder inyectarlas donde se necesiten. Tiene las mismas funcionalidades que las etiquetas @Service, @Controller y @Repository.

Configuración en Spring

@Configuration es la anotación encargada de definir una clase de configuración para el framework.

@Bean es la anotación que marca como bean cada uno de los métodos de tal forma que esten disponibles para Spring.
