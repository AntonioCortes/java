Operadores aritméticos
Simbolo Funcionalidad
+ Suma dos valores
- Resta dos valores
* Multiplica dos valores
/ Division
% Resto
+= Suma y asigna el valor
-= Resta y asigna el valor
*= Multiplica y asigna el valor
/= Divide y asigna el valor
%= realiza el modulo y asigna el valor
Operadores Relacionales
Simbolo Funcionalidad
== Comprueba si los datos son iguales
!= comprueba si lod datos son diferentes
< Comprueba que el primer dato sea menor que el segundo
> Comprueba que el primer dato sea mayor que le segundo
<= Comprueba que el primer dato sea menor o igual que el segundo
>= Comprueba que el primer dato sea mayor o igual que el segundo
! Invierte el valor de la expresión
& (AND) Compara dos expresiones y devuelve TRUE si ambas se cumplen
&& Compara dos expresiones y devuelve TRUE si ambas se cumplen. Si la primera condicion es FALSE no se evalua la segunda.
| (OR) Compara dos expresiones y devuelve TRUE si al menos una se cumple
|| (OR) Compara dos expresiones y devuelve TRUE si al menos una se cumple. Si la primera condicion es TRUE no se evalua la segunda.
^ (XOR) Compara dos expresiones y devuelve TRUE si ambos resultados son distintos.
Operadores Unarios
Simbolo Funcionalidad
++ Incrementa el valor de una variable
-- Decrementa el valor de una variable

Preincremental y Postdecremental:


int a = 0;
int b = a++;
//Se asigna el valor a "b" y luego se incrementa el valor de "a"
//a = 1
//b = 0

int c = 0;
int d = ++c;
//Se incrementa el valor de "c" y luego se asigna el valor a "d"
//c = 1
//d = 1
