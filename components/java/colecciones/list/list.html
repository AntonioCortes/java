Listas

Una colección es una agrupación de objetos sin tamaño fijo

Se puede añadir y eliminar objetos de una colección dinámicamente

Para gestionar colecciones disponemos de clases e interfaces específicas en java.util

Tipos:

  • Listas
  • Tablas
  • Conjuntos

Cada elemento tiene una posición asociada a partir del orden de llegada, siendo 0 la posición del primero.

Las listas implementan la interfaz List, que a su vez implementa Collection.

Son colecciones de tipo genérico (preparadas para admitir cualquier objeto Java).

La principal clase de colección es ArrayList.

Creacion de listas

Como instancias de ArrayList:


List<Integer> enteros = new ArrayList<>();

//...

A partir del método asList de Arrays:

NOTA: Lista de tamaño FIJO, no admite inserción ni eliminación.


List<Integer> enteros = arrays.asList(6,2,54,678,1);

//...

Mediante método de factoría y coptOf() de List:

NOTA: INMUTABLES, no admiten la modificación e inserción de elementos, ni valores null.


//...
List<Integer> enteros = List.of(40,5,76,1);

List<Integer> copia = arrays.copyOf(enteros);

Recorrido de una lista

Se puede recorrer con un bucle for estándar:


List<String> nombres = List.of("Juan","Juana","Jake");
//muestra el contenido de la lista de nombres
for(int i=0; i < nombres.size(); i++){
    System.out.println(nombres.get(i));
}

También mediante un for each:


List<String> nombres = List.of("Juan","Juana","Jake");
//misma función que el bloque anterior
for(String s:nombres){
    System.out.println(s);
}

Método forEach(lambda)


List<String> nombres = List.of("Juan","Juana","Jake");
Stream st = Stream.of(nombres);
//muestra todos los elementos
st.forEach(n->System.out.println(n));

Usando un Iterador


List<Integer> list = new ArrayList<>(List.of(1,2,3,4,5,6,7,8,9));
Iterator<Integer> iterator = list.iterator();
while (iterator.hasNext()){
    System.out.println(iterator.next());
}

Principales métodos para listas

Métodos para List
Método Retorno Funcionalidad
list.add(T dato) boolean Añade el dato a la colección y lo coloca al final de la misma. "T" representa el tipo indicado al crear el objeto de colección.
list.add(int pos,T dato) boolean Añade el elemento en la posición indicada, desplazando hacia adelante los que se encuentren en dicha posición.
list.set(int pos,T dato) T Sustituye el elemento existente en la posición indicada por el nuevo dato suministrado como parámetro. Devuelve el elemento sustituido
list.size() int Devuelve el total de elementos de la colección.
list.get(int pos) T Devuelve el elemento que ocupa la posición indicada. Si la posición es menor que 0 o mayor o igual que size(), se producirá una excepción.
list.remove(int pos) T Elimina el elemento que ocupa la posición indicada y lo devuelve. Si la posición es menor que 0 o mayor o igual que size(), se producirá una excepción.
list.remove(Object ob) boolean Elimina el elemento indicado en caso de que exista. Devuelve true si dicho elemento estaba presente en la colección. Si hay más de un elemento, elimina la primera ocurrencia

Revisión de conceptos


List<Integer> nums=List.of(11,22,31,10);
nums.set(0,nums.get(1)); //EXCEPCION EN ESTA LINEA
System.out.println(nums.get(2)-nums.get(0));

Se producirá una excepción en la segunda instrucción, ya que se trata de una lista inmutable y no se puede modificar su contenido


List<Integer> nums=new ArrayList(List.of(6,11,22,31,10,5,7)); //1
for(int k=0;k<nums.size();k++) {
    if(nums.get(k)%2==0) {
    nums.remove(k);
    }
}
System.out.println(nums);

La primera instrucción compila bien, es posible crear un ArrayList a partir de un Collection. El ArrayList es mutable, por lo que se puede modificar su contenido y la línea 2 es correcta.
