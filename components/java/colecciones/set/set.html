Set

Clase diseñada para trabajar con conjuntos.

Los elementos no tienen posición ni clave asociada, si bien cada elemento es único, no se pueden repetir.

Emplea internamente los métodos equals y hashcode para determinar la igualdad de objetos.

Los conjuntos implementan la interfaz Set, que es de tipo genérico.

La principal clase de conjuntos es HashSet.

Forma de crear conjuntos o Set


//MUTABLES
Set<String> nombres = new HashSet<>();


//INMUTABLES, mo admiten la eliminación,
//modificación e inserción de elementos. 
//Tampoco admite valores null
Set<String> nombres = Set.of("Juan","Jake");
Set<String> nuevos = Set.copyOf(nombres)

Métodos de HashSet

Métodos para HashSet
Método Retorno Funcionalidad
set.add(T dato) boolean Añade el dato a la colección si no está presente. Devuelve true si lo ha podido añadir.
set.remove(T dato) boolean Añade el elemento en la posición indicada, desplazando hacia adelante los que se encuentren en dicha posición.
set.size() int Devuelve el tamaño de la colección.
set.contains(Object ob) boolean Devuelve true si el objeto existe en la colección.

Ejemplos de métodos de HashSet


HashSet<String> dias =new HashSet<>();
dias.add("lunes");
dias.add("martes");
dias.add("lunes"); //no lo añade

Recorrido de un conjnto

Un conjunto puede ser recorrido con un for each


for(String s:dias){
    System.out.println(s); //muestra cada elemento
}

También dispone del método forEach

Método forEach(lambda)


Set<String> nombres = Set.of("Juan","Juana","Jake");
Stream st = Stream.of(nombres);
//muestra todos los elementos
st.forEach(n->System.out.println(n));
