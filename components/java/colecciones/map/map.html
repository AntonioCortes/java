Map

  • Cada elemento tiene asociada una clave única
  • No hay un orden o posición
  • Las tablas implementan la interfaz Map
  • Tanto el tipo de la clave como del valor son genéricos
  • La principal clase de colección es HashMap Map

Creación de tablas


//MUTABLES
Map<Integer,String> contactos = new HashMap<>();


//INMUTABLES, no admiten la eliminación, modificación e
//inserción de elementos. No admiten ni claves ni valores null
//Método factoría:
Map<Integer,String> contactos = Map.ofEntries(
    Map.entry (123, "Juan"),
    Map.entry (300, "Ana"),
    Map.entry (500 ,"Olivia")
);

//Método copyOf:
Map<Integer , String > Map.copyOf(contactos);

Métodos de HashMap

Métodos de HashMap
Método Retorno Funcionalidad
hashMap.put(K clave, T dato) T Añade el dato a la colección y le asocia la clave indicada como primer parámetro. Si ya existiera esa clave, el valor existente será sustituido por el nuevo. NOTA: el método admite claves null.
hashMap.putIfAbsent(K clave, T dato) T Añade la entrada en caso de que la clave no exista o tenga asociado el valor null.
hashMap.get() T Devuelve el dato que tenga asociada dicha clave. Si no hay ninguno, devolverá null.
hashMap.size() int Devuelve el tamaño de la colección.
hashMap.remove(Object clave) T Elimina el dato que tenga dicha clave asociada y devuelve el elemento eliminado.
hashMap.containsKey(K key) boolean Indica si hay algún elemento en la colección con dicha clave asociada.
hashMap.containsValue(T valor) boolean Indica si el elemento está presente en la colección.

Recorrido de una tabla

Se pueden utilizar los siguientes métodos:

  • Collection<T> values(). Devuelve una colección solo con los valores. Puede utilizarse para recorrer el conjunto de valores con un for each.
    • 
    Collection<String> datos = tabla.values();
    for(String s:datos){
        System.out.println(s); //muestra cada elemento
    }
  • Set<K> keySet(). Devuelve el conjunto de claves de la tabla. Se puede recorrer con un for each.
    • 
    Map<Integer, String> map = new HashMap<>(Map.of(1,"Juan",23,"Jake"));
    Stream.of(map.keySet()).forEach(k-> System.out.println(k));
  • Método forEach(lambda)
    • 
    Map<Integer, String> map = new HashMap<>(Map.of(1,"Juan",23,"Jake"));
    Stream.of(map.keySet()).forEach(k-> System.out.println(k));

TreeMap

Similar a HashMap , con la diferencia de que los objetos están ordenados por la clave: 


TreeMap<Integer,String> tabla=new TreeMap<>();       
tabla.put(200, "dato1");
tabla.put(400, "dato2");
tabla.put(100,"dato3");
for (String val:tabla.values()){
    System.out.print(val); // dato3 dato1 dato2
}
