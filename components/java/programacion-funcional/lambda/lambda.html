Expresiones Lambda

Una expresión lambda es la implementación de una interfaz funcional.


@FunctionalInterface
public interface myPersonalFunctionalInterface<T> {
        
    String mySuperAbstractMethod(T t);
    
}


//...

myPersonalFunctionalInterface l = (String p)->(p+" ");

//...

Los parámetros pueden especificar o no el tipo

NOTA: A partir de dos o mas parámetros, o si el único parámetro tiene especificado su tipo, es necesario poner paréntesis.

NOTA: Si se especifica el return en la devolución es necesario poner paréntesis a aquello a devolver.

Inferencia de tipos

Es posible inferir el tipo en los parámetros de las expresiones lambda:

Aunque no se puede combinar inferencia de tipos y tipos específicos en una misma expresión:


(var a)->System.out.println(a)
    
(var a, int c)->m(a) //ERROR DE COMPILACION
    
(@NotNull var c)->m(a) //OK
(@NotNull c)->m(a) //ERROR DE COMPILACION
