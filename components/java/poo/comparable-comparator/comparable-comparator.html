Interfaces: Comparable y Comparator

Comparator

Comparator, es una interfaz que se implementa en clases que nos ayudan o que están destinadas a comparar aspectos de otras clases. Por lo que la clase donde se implemente dicha interfaz, no se utilizará en la comparación, si no que es la clase destinada a realizar comparación. Para realizar la comparación sobrescribe el método compare().

Ejemplo:


// Class to compare Movies by ratings
class RatingCompare implements Comparator<Movie>
{
    @Override
    public int compare(Movie m1, Movie m2)
    {
        if (m1.getRating() < m2.getRating()) return -1;
        if (m1.getRating() > m2.getRating()) return 1;
        else return 0;
    }
}

Comparable

Al contrario que Comparator, la clase que implemente la interfaz Comparable adquiere la funcionalidad de poder ser comparada con otros objetos. Para ello sobrescribe el método compareTo().

Ejemplo:


class Movie implements Comparable<Movie>{
    
    private double rating;
    private String name;
    private int year;
        
    // Used to sort movies by year
    public int compareTo(Movie m){
        return this.year - m.year;
    }
}
