Métodos para Arrays
Método Retorno Funcionalidad
unArrayDeque.push([E]) void Añade el elemento que recibe por argumento en la posición 0 del array y desplaza el resto de elementos.
unArrayDeque.add([E]) boolean Añade el elemento que recibe por argumento en la última posición del array, es decir, al final. NOTA: todo lo contrario que el método push()
unArrayDeque.remove() E Elimina el elemento en la posición 0 del array y lo devuelve.
unArrayDeque.peek() E Devuelve el elemento en la posicion 0 del array.
unArrayDeque.size() int Devuelve el tamaño del array
