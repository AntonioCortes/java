Tipos de datos

NOTA: "var" es el único tipo de dato que se puede utilizar como nombre de una variable, sin que esta se considere palabra reservada

El compilador interpreta que los numeros son enteros los que no llevan decimales Aquí hay error de compilacion ya que se interpreta que es un entero y se pasa de grande => long numLong = 10000000000 Para que no ocurra añadir que es un Long con L => long numLong = 10000000000L El compilador interpreta que los numeros son doubles los que llevan decimales para que le compilador entienda que lo que metes es float indicar la F al final del numero: float numFloat = 10.5F de no hacer esto habría que hacer un cast float numFloat = (float) 10.5

NOTA: esto se puede => numLong = 1_000_000L

NOTA: esto se puede => numLong = 1_000_____000L

NOTA: esto NO se puede => numLong = _1_000_000L
