Patrón Transfer Object Assembler

El patrón Transfer Object Assembler es un patrón de la arquitectura Java EE que se encarga de recolectar datos de varias fuentes y ensamblarlos en un objeto de transferencia completo para la vista.

Propósito

Proveer a la vista de todos los datos necesarios en un solo objeto optimizado.

Caso de Uso Real

Un 'DashboardDTO' que contiene datos de usuario, últimas órdenes y mensajes, recogidos de 3 servicios diferentes.

Implementación


class DashboardAssembler {
    public DashboardDTO createDashboard(Long userId) {
        User user = userService.getUser(userId);
        List orders = orderService.getRecent(userId);
        List msgs = msgService.getUnread(userId);
        return new DashboardDTO(user, orders, msgs);
    }
}
