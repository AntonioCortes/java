Patrón Prototype

El patrón Prototype es un patrón creacional que permite copiar objetos existentes sin que el código dependa de sus clases.

Propósito

Crear nuevos objetos clonando una instancia existente (prototipo). Reduce el costo de creación.

Caso de Uso Real

Clonación de configuraciones complejas o personajes en un juego (spawning enemies).

Implementación


abstract class Shape implements Cloneable {
    public int x, y;
    
    public Shape() {}
    
    public Shape(Shape target) {
        if (target != null) {
            this.x = target.x;
            this.y = target.y;
        }
    }
    
    public abstract Shape clone();
}

class Circle extends Shape {
    public int radius;

    public Circle(Circle target) {
        super(target);
        if (target != null) {
            this.radius = target.radius;
        }
    }

    @Override
    public Shape clone() {
        return new Circle(this);
    }
}
