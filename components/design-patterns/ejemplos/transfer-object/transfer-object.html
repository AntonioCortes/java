Patrón Transfer Object (DTO)

El patrón Transfer Object (DTO) es un patrón de la arquitectura Java EE que transporta datos desde la capa de negocio a la de presentación en una sola llamada.

Propósito

Reducir el número de llamadas remotas transportando múltiples datos a la vez.

Caso de Uso Real

Enviar datos de un usuario (nombre, email, roles) a la UI en un solo objeto JSON/Java.

Implementación


class UserDTO implements Serializable {
    public String username;
    public String email;
    // Getters, Setters, Constructor
}
