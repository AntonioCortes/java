Anotación @Builder

Esta anotación permite aplicar el patrón "Builder" a una clase sin tener que escribir el código. También puede ser usado sobre métodos para trabajar con clases inmutables.

Ejemplo:


    @Builder
    public class Customer {
        private Integer id;
        private String firstName;
        private String lastName;
        private String email;

    }

    Customer customer = Customer.builder()
                .firstName("Luffy")
                .lastName("Monkey")
                .email("luffy_d_monkey@grandline.com")
                .build();

Atributos

La anotación tiene como atributo (builderMethodName = "") y su valor predefinido es "builder"

Ejemplo:


        @Builder(builderMethodName = "buildingTheThing")
        public class Customer {
            private Integer id;
            private String firstName;
            private String lastName;
            private String email;

    
        }
    
        Customer customer = Customer.buildingTheThing()
                    .firstName("Luffy")
                    .lastName("Monkey")
                    .email("luffy_d_monkey@grandline.com")
                    .build();
